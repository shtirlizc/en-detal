Проектирование

Наше предприятие имеет возможность разработки проектной документации на различное технологическое оборудование по требованиям:

  1. ГОСТ Р 52630-2012 "Сосуды и аппараты стальные сварные. Общие технические требования".
  2. ГОСТ 31838-2012 "Аппараты колонные. Технические требования".
  3. ГОСТ 53677-2009 "Нефтяная и газовая промышленность. Кожухотрубчатые теплообменники. Технические требования".

В комплекте проектной документации предоставляются чертежи (возможна полная деталировка), спецификации, пояснительные записки, технологические расчеты, расчеты на прочность, руководство по эксплуатации.

Проектная документации разрабатывается в следующих форматах:

  1. Техническое задание для заказчика.
  2. Расширенное техническое задание для тендеров.
  3. Технический проект для заказчика.
  4. Технический проект с деталировкой для заводов изготовителей. ики, штуцеры).

При разработке проектной документации технологического оборудования используется следующее программное обеспечение:

  1. AutoCad
  2. Autodesk Inventor
  3. Autodesk Simulation Multiphysics
  4. Solidworks
  5. Solidworks flow simulation
  6. PVP-Design 4.0

Расчеты на прочность технологического оборудования выполняются как по требованиям ГОСТ Р 52857.1-11-2007, ПНАЭ Г-7-002-86 с использованием рекомендованных методик и методов конечных элементов (МКЭ).

Опыт проектирования:

  1. Горизонтальных и вертикальных емкостных аппаратов (емкости, сборники, сепараторы, отстойники, деаэраторы).
  2. Вертикальные колонные аппараты (ректификационные колонны, абсорберы, реакторы).
  3. Теплообменные аппараты (кожухотрубчатые, испарители с паровым пространством и без).
  4. Нестандартные опоры аппаратов и трубопроводов (неподвижные, подвижные, катковые, хомутовые).
  5. Нестандартные соединительные детали трубопроводов (отводы, гибы, переходы, тройники, штуцеры).
